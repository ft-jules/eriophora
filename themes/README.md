# Thèmes
Un dossier par thème. Fichiers requis :
- `wallpaper.png` (ou .jpg)
- `colors-hypr.conf` · `colors-kitty.conf` · `colors-waybar.css` · `colors-rofi.rasi` · `colors-mako.conf`

Bascule : `theme-set <nom>` (symlink `~/.config/themes/current` + reload).
Catalogue & palettes : vault → `Thèmes - Arch`.
À venir : vhs, archipel, nihei, vermis (wallpapers dans le vault, 98_ressources).
